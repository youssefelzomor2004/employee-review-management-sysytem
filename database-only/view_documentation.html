Stores user accounts, roles, and organizational hierarchy

📋 Business Requirements

👤 User Management

  • 4 user roles: Admin, Employee, Manager-Employee, Highboard Manager
  • New users default to PENDING status
  • Unique usernames required
  • Admin can suspend accounts
  • Default admin user created on setup

📊 Performance Reviews

  • Basic (All): Punctuality, Teamwork, Productivity
  • Extended (Managers): Leadership, Vision, Managerial Skills
  • All scores: 0-10 scale
  • Anonymous reviews supported
  • Comments optional

🎯 Goal Management

  • Self-created or manager-assigned goals
  • Status tracking: ACTIVE → COMPLETED
  • Users can delete own-created goals
  • Deleted with employee (CASCADE)

🎓 Training & Development

  • Central course catalog
  • Managers recommend courses to employees
  • Tracks who recommended what
  • 5 default courses pre-loaded
